# Memory: tech/crawl/comprehensive-engine-v4-fr
Updated: 2026-04-13

Moteur de Crawl (v4) : 1. Découverte hybride : Sitemap XML + Spider.cloud/Firecrawl map + **CMS content discovery** (interrogation automatique de tous les CMS connectés via `cmsContentScanner` : WordPress, Shopify, IKtracker, etc.). Cette triple source garantit la détection des pages dynamiques (SPA React/Vite) non référencées dans le sitemap. 2. Précision : Le bouton 'Analyser' (ou 'Lancer le crawl') est désactivé tant que la détection du nombre de pages est en cours pour éviter les lancements prématurés. 3. Historique : Chaque session crée une nouvelle ligne dans 'site_crawls' (pas de mise à jour) avec purge manuelle possible. 4. Reporting : 'Gap Context' expliquant l'écart entre URLs détectées et crawlées. 5. Infrastructure : Isolation par 'user_id' et normalisation d'URL. 6. Résilience : Bouton 'Stop' et timeout global 120s. 7. UI : Utilisation de 'caret-foreground' pour garantir la visibilité du curseur de texte dans le champ URL.
