# Memory: features/console/reports-fr

## Sous-menu Rapports (Console)
- Onglet "Rapports" à droite de GMB dans le sous-menu Console.
- Menu vertical gauche avec les URLs trackées (identique à "Mes Sites").
- Par onglet de site : système de dossiers imbriqués (dossier dans dossier) + rapports dans dossiers.
- Actions dossiers : créer, renommer, supprimer (avec avertissement du nombre de rapports contenus).
- Actions rapports : renommer, supprimer, télécharger.
- Éléments supprimés rangés dans un menu déroulant "Archives" en bas.
- Données stockées dans la table `site_reports` avec `folder_path` et `is_archived`.
