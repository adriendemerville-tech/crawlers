# Audit Code — Audit Compare (audit-compare)
Date : 2026-07-08
Auditeur : Senior Crawlers Auditor
Perimetre : Backend (Edge), Frontend (React), Tables.

## Synthese Executive
L'audit revele une fonctionnalite Battle Card puissante mais souffrant d'une architecture monolithique critique (1062 l. backend, 1677 l. frontend) et de fuites de couts LLM. Le contournement de l'AI Gateway et l'absence de rate-limit IP pour les appels payants (DataForSEO) sont les priorites P0.

## Findings P0/P1

### #149 · P0 · 2j — Fuite de couts LLM et AI Gateway bypass
Le backend utilise fetch directement vers OpenRouter au lieu de aiGatewayFetch.
Fix : Migrer generateSeedsWithAI, analyzeSite et runCrossComparison vers aiGatewayFetch.

### #150 · P1 · 1j — Absence de Rate-Limit IP (Lead Magnet Drain)
Les appels payants DataForSEO sont declenchables sans restriction IP par des utilisateurs anonymes.
Fix : Implementer ipRateLimiter sur le handler principal avant les appels payants.

### #151 · P1 · 1.5j — Monolithe Frontend et Re-renders massifs
AuditCompare.tsx (1677 lignes) melange logique, types, i18n et composants complexes non memoises.
Fix : Splitter en sous-composants dans features/audit-compare/components/ (CompareForm, RadarBattle, etc.).

### #152 · P1 · 1j — SSRF et manque de validation sur extractPageMetadata
La fonction extractPageMetadata fetch des URLs utilisateurs sans validation de securite.
Fix : Utiliser assertSafeUrl de _shared/ssrf.ts et stealthFetch pour les requetes sortantes.

## Findings P2

### #153 · P2 · 0.5j — TTL Cache inadapte (Hardcoded 2h)
Le cache est limite a 2h alors que les donnees backlinks et keywords sont stables a long terme.
Fix : Passer a un TTL de 24h minimum, idealement 7j pour les domaines non-frais.

### #154 · P2 · 1j — Monolithe Backend (1062 l.)
Le fichier index.ts contient trop de logique metier interne non mutualisee.
Fix : Deporter la logique dans _shared/audit-compare/ ou utiliser les utilitaires partages.

### #155 · P2 · 0.5j — Detection SPA inefficace
Le fallback Browserless est declenche sur un simple check de longueur de texte.
Fix : Utiliser isSpaDetected (shared) base sur les signatures de frameworks.
