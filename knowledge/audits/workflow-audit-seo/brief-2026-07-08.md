# Audit Brief — Workflow Audit SEO
Date : 2026-07-08 · Auditeur : Lovable · Baseline : delta vs `knowledge/audits/expert-seo/brief-2026-07-08.md` (Vague 3 Partie 5)

## 1. Objectif métier
Fournir un audit SEO on-page + technique + contenu fiable et actionnable par persona, cohérent avec le reste de l'écosystème (Copilot, Workbench, Autopilot).

## 2. Cas d'usage principaux
- Free → lance un audit URL unique → score global + top 5 actions
- Premium → audit multi-URL + export → plan éditorial injecté dans Workbench
- Pro Agency → audit d'un site client + partage rapport → deltas vs concurrents (SERP benchmark)
- Admin/Fantômas → replay audit avec overrides LLM

## 3. Logique métier clé
- Backend : `audit-expert-seo`, `expert-audit`, `render-page`, `crawl-multi-pages`
- Enrichissements : `computeSeoScoreV2`, EEAT v3, Priority Score, self-crawlability sprint 4
- Dépendances : SERP benchmark, keyword_universe SSOT, matrix system v2
- Routage LLM : `aiRouter` (override Groq possible)
- Gating : plan (URL count, export), quotas
- Contraintes mémoire : `shared/seo-scoring-v2-fr`, `audit/eeat-scoring-v3-fr`, `audit/priority-and-bot-standards-fr`

## 5. Benchmark concurrentiel
- **SEMrush Site Audit** — 130+ checks, crawl JS, très exhaustif mais peu prescriptif
- **Ahrefs Site Audit** — bon graphe interne, faible sur on-page contenu
- **Screaming Frog** — outil desktop, roi du technique brut
- **SEObility / Sitechecker** — français, plus léger, UX simple
- Barre : atteindre l'exhaustivité SEMrush sur le technique + surpasser Ahrefs sur le prescriptif (Workbench + Autopilot) grâce à la boucle IA
