# Audit Brief — audit-expert-seo (+ expert-audit)
Date : 2026-07-08 · Auditeur : Lovable · Baseline : première itération

## 1. Objectif métier
Fournir à l'utilisateur Crawlers un **rapport GEO/SEO expert one-shot** sur une URL (audit gratuit d'appel + version enrichie Premium) qui diagnostique visibilité IA, chunkability, E-E-A-T, fan-out, maillage, hallucinations, GMB et propose un plan d'action correctif. C'est la **porte d'entrée conversion** du produit (freemium → payant).

## 2. Cas d'usage principaux
- **Visiteur anonyme** → colle une URL sur la home → reçoit ExpertAuditDashboard (score global + 20+ cards) → email gate → conversion trial.
- **Free** → 1 audit expert / mois, sans plan d'action détaillé ni corrections HTML.
- **Premium** → audits illimités, plan d'action, corrections HTML injectables, comparaison avant/après (`audit-compare`, `measure-audit-impact`).
- **Pro Agency** → audits par client, snapshots, PDF, comparaison multi-sites.
- **Admin** → voit tous les audits, peut relancer, corriger les hallucinations, ajuster le scoring EEAT.

## 3. Logique métier clé
- **Deux edge functions** :
  - `audit-expert-seo` (2 338 l.) — audit expert nouveau, appelé par la home et le dashboard
  - `expert-audit` (2 987 l.) — legacy plus complet, encore actif
  - Chevauchement suspecté → à statuer (fusion ? déprécation ?)
- **Dépendances** :
  - LLM : Lovable AI (Gemini 2.5) via `aiGatewayCall` — **à vérifier**, `audit-compare` figure dans la liste des 3 offenders LLM hors gateway (Vague 1)
  - APIs externes : DataForSEO (SERP + backlinks), Browserless (rendering), Google PageSpeed, GSC (si connecté), GMB Places
  - Tables : `audits`, `audit_cache`, `audit_raw_data`, `audit_impact_snapshots`, `hallucination_corrections`, `page_priority_scores`, `serp_intent_analyses`
  - Mémoires applicables : `eeat-scoring-v3`, `fan-out-detection-strategy`, `strategic-audit-engine-v5`, `combo-abc-allocation`, `priority-and-bot-standards`, `felix-raw-audit-access`
- **Gating** : freemium (1/mois anonyme via `useDemoSafeFetch`), Premium (illimité + plan d'action + corrections), Pro Agency (multi-clients).
- **Scoring** : score global 0-100 pondéré sur ~15 axes (GEO, technique, contenu, autorité, local).

## 5. Benchmark concurrentiel
- **Ahrefs Site Audit** — 170+ issues, crawl complet, très technique/SEO classique, aucun signal IA/GEO. Force : profondeur technique. Faiblesse : ne parle pas aux LLM.
- **Screaming Frog** — desktop, brut, expert only. Force : contrôle total. Faiblesse : pas de scoring, pas de plan d'action, pas de GEO.
- **Semrush Site Audit** — 140+ checks + Position Tracking, "AI Toolkit" récent mais superficiel. Force : ubiquité. Faiblesse : GEO gadget.
- **Thot SEO / SEO Rank Monitor** (FR) — SEO classique + surveillance. Faiblesse : zéro GEO/AI.
- **Mentionnable** — mentions LLM (ChatGPT/Perplexity/Claude). Force : niche GEO pure. Faiblesse : pas de plan d'action technique, pas d'audit page.
- **Barre à dépasser** : **seul audit qui unifie SEO classique + GEO (fan-out, chunkability, E-E-A-T, hallucinations LLM) + plan d'action + corrections HTML prêtes à injecter**. Différenciateur : boucle diagnostic → correction → mesure d'impact (`measure-audit-impact` + `snapshot-audit-impact`), inexistante chez les concurrents.
