# Audit Brief — Marina (outreach B2B / prospecting audit)
Date : 2026-07-08 · Auditeur : Lovable · Baseline : première itération complète

## 1. Objectif métier
Audit d'entreprise cible pour outreach B2B (SEO/GEO/social/tech) → livrable partageable en white label, injecté dans workflow prospection LinkedIn.

## 2. Cas d'usage principaux
- Pro Agency → saisit domaine prospect → Marina génère audit multi-axes → export PDF white label
- Pro Agency → pipeline outreach LinkedIn automated (mémoire `outreach-pipeline`)
- Copilot Stratège → skill `marina_prospect_audit` → résultat dans vector memory pour rappel conversationnel
- Admin → surveille quotas + qualité livrables Marina

## 3. Logique métier clé
- Backend : `marina-module` + fonctions audit prospect (SEO score, GEO score, backlinks, GMB, social presence, tech stack)
- White labeling : override branding sur PDF/rapports partagés
- Pipeline outreach : LinkedIn automated (contraintes OAuth social custom)
- Alignement Vague 4 : les fonctions Marina d'audit DOIVENT réutiliser les modules audités aux briefs #1 (SEO), #2 (GEO), #5 (Spiral) — pas de duplication de logique
- Contraintes mémoire : `features/prospecting/marina-module-fr`, `features/prospecting/outreach-pipeline-fr`, `constraints/social-auth-limitations-fr`, `features/pro-agency/team-sharing-fr`
